# DASHBOARD SPEC — Forge-Stack-A1 Front End
# Specification for the Sentinel Task Control dashboard

## COMPONENTS

### Timer Block
- Pomodoro: 25 min (yellow)
- Break: 5/10/15 min (green)
- Auto-transition: break ends → task appears

### Task Block
- One task displayed at a time
- One next step only
- Prev / Next navigation
- Progress bar (step X of Y)

### Status Tags
- DO FIRST (red) — urgent + important
- SCHEDULE (blue) — important, not urgent
- DELEGATE (yellow) — urgent, not important
- ELIMINATE (grey) — neither

### Session Header
- Shannon energy level
- Active protocol indicators
- Zone status (GREEN/YELLOW/RED)

## EISENHOWER MATRIX INTEGRATION
Tasks routed by quadrant:
- Q1 (Urgent + Important) = DO FIRST → GREEN zone
- Q2 (Not Urgent + Important) = SCHEDULE → YELLOW zone
- Q3 (Urgent + Not Important) = DELEGATE
- Q4 (Not Urgent + Not Important) = ELIMINATE

### State Variables
- `activeNodes` — Set of activated node IDs
- `systemStatus` — string (INITIALIZING / ONLINE / QUANTUM ACTIVE)
- `pulsePhase` — float (0–2π, drives animations)
- `protocolsActive` — bool (all 11 nodes active)
- `latticeExpanded` — bool (Platonic solid overlay visible)
- `quantumFlux` — bool (particle ring active at center)
- `userEngagement` — int (increments on node click)
- `audioPlaying` — bool (optional audio toggle)

### Dashboard Integration
The QNF Visualization Panel is an optional second panel alongside the Sentinel Task
Control dashboard. It is:
- Launched separately or embedded as a collapsible sidebar
- Driven by the same zone state (GREEN/YELLOW/RED) as the Session Header
- CORE NEXUS node maps to active zone; pulse speed reflects cognitive load (1–10)
- Does NOT replace the Pomodoro/Eisenhower task workflow — it visualizes system state
